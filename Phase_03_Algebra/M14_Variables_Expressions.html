What is a Variable?

A variable is a symbol (usually a letter) that represents an unknown or changing number.

Instead of writing specific numbers, we use letters like x, y, a, b to represent numbers that can change.

Think of it as: "A box that holds a number"

What is an Expression?

An expression is a combination of variables, numbers, and operations (like +, -, √ó, √∑).

It's a mathematical phrase (not a complete sentence with = sign).

Expression examples: 2x + 5, 3a - 7, n/2 + 10

Parts of an Expression:

Expression: 3x + 5

3 = Coefficient (the number multiplied by the variable)
x = Variable (the unknown)
+ = Operation (plus sign)
5 = Constant (a fixed number)

Terms in an Expression:

A term is a single part of an expression separated by + or - signs.

Expression: 2x + 5 - 3y

Simplifying Expressions

Simplifying means combining like terms to make the expression shorter and easier to use.

Example 1: Combining Like Terms

Simplify: 3x + 5x

1 Identify like terms:
3x and 5x are like terms (both have x)
2 Add the coefficients:
3 + 5 = 8
3 Keep the variable:
Result: 8x
‚úì 3x + 5x = 8x

Example 2: Combining Multiple Terms

Simplify: 2x + 7 + 5x + 3

1 Group like terms:
(2x + 5x) + (7 + 3)
2 Combine x terms:
2x + 5x = 7x
3 Combine number terms:
7 + 3 = 10
4 Write simplified expression:
7x + 10
‚úì 2x + 7 + 5x + 3 = 7x + 10

Example 3: With Subtraction

Simplify: 9a - 3a + 5

1 Combine a terms:
9a - 3a = 6a
2 Keep the constant:
6a + 5
‚úì 9a - 3a + 5 = 6a + 5

Operations with Variables (Order of Operations)

When evaluating expressions with variables, follow PEMDAS/BODMAS:

Parentheses / Brackets
Exponents / Orders
Multiplication & Division (left to right)
Addition & Subtraction (left to right)

Example: Order of Operations

Evaluate: 2 + 3 √ó x, when x = 4

1 Replace x with 4:
2 + 3 √ó 4
2 Multiply FIRST (before adding):
3 √ó 4 = 12
3 Then add:

‚úì Answer: 14

Note: If we did addition first: 2 + 3 = 5, then 5 √ó 4 = 20 (WRONG!)
