Theory: What is Absolute Value?

üìè ABSOLUTE VALUE

Definition: The DISTANCE from a number to ZERO on a number line. Always POSITIVE!

Simple Words: How FAR is it from zero? Ignore the direction!

Notation: |x| means "absolute value of x"
Example: |5| = 5, |-5| = 5
Goal: Always a POSITIVE answer

Key Idea: Absolute value measures HOW FAR, not WHICH DIRECTION
