What is Central Tendency?

Central Tendency describes the center or typical value of a dataset.

Three main measures: Mean (Average), Median (Middle), Mode (Most Frequent)
üìà Mean The average of all values

üìç Median The middle value

üîÑ Mode Most frequent value

Mean (Average)

The Mean is the sum of all values divided by the count of values.
Most commonly used measure of central tendency.
Mean = Sum of all values √∑ Number of values

Mean = (x‚ÇÅ + x‚ÇÇ + x‚ÇÉ + ... + x‚Çô) √∑ n

üìù Example: Find the Mean

Data: 5, 10, 15, 20, 25
1Add all values:
5 + 10 + 15 + 20 + 25 = 75
2Count the values:
n = 5
3Divide:
Mean = 75 √∑ 5 = 15
Mean = 15

üìù Example 2: Test Scores

Data: 85, 90, 78, 92, 88, 95
1Sum: 85+90+78+92+88+95 = 528
2Count: n = 6
3Mean: 528 √∑ 6 = 88
Mean = 88
üí° Note: Mean is affected by extreme values (outliers)!
Example: {10, 10, 10, 10, 1000} ‚Üí Mean = 208 (misleading!)

Median (Middle Value)

The Median is the middle value when data is sorted.
Not affected by extreme values - better for skewed data!
Step 1: Sort the data (ascending order)
Step 2: Find the middle value

Odd count ‚Üí Middle number
Even count ‚Üí Average of two middle numbers

Example 1: Odd Number of Values

üìù Find the Median

Data: 7, 3, 9, 1, 5
1Sort: 1, 3, 5, 7, 9
2Count: 5 values (odd)
3Find middle: 1, 3, 5, 7, 9
Median = 5

Example 2: Even Number of Values

üìù Find the Median

Data: 4, 8, 2, 6
1Sort: 2, 4, 6, 8
2Count: 4 values (even)
3Find two middle values: 2, 4, 6, 8
4Average them: (4 + 6) √∑ 2 = 5
Median = 5

Mode (Most Frequent)

The Mode is the value that appears most frequently.
A dataset can have no mode, one mode, or multiple modes.
Count how many times each value appears
The value with highest count = Mode

üìù Example 1: Single Mode

Data: 2, 4, 4, 4, 5, 6, 7
1Count each:
2‚Üí1, 4‚Üí3, 5‚Üí1, 6‚Üí1, 7‚Üí1
2Most frequent: 4 appears 3 times
Mode = 4

üìù Example 2: Two Modes (Bimodal)

Data: 1, 2, 2, 3, 3, 4
1Count: 2‚Üí2 times, 3‚Üí2 times
Modes = 2 and 3 (Bimodal)

üìù Example 3: No Mode

Data: 1, 2, 3, 4, 5
All values appear exactly once
No Mode

Comparison: When to Use Each?

Measure Best When Weakness
Mean Data is symmetric, no outliers Affected by extreme values
Median Data has outliers or is skewed Ignores actual values
Mode Categorical data, finding most common May not exist or not unique

üìù Complete Example

Data: 10, 20, 20, 30, 40, 50, 100
Mean: (10+20+20+30+40+50+100) √∑ 7 = 270 √∑ 7 = 38.57
Median: 10, 20, 20, 30, 40, 50, 100 = 30
